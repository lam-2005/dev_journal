# DevJournal
 
Nền tảng chia sẻ kiến thức, lưu trữ kinh nghiệm và ghi lại hành trình phát triển mỗi ngày của lập trình viên.

## Website
  
Truy cập website tại: [DevJournal](https://dev-journal-nine.vercel.app/)  
> **Lưu ý**: Khi truy cập website, do API backend được host trên server miễn phí nên có thể mất một chút thời gian để khởi tạo lần đầu. Vui lòng chờ vài phút nếu thấy website tải lâu

## Chức năng chính  

- **Đăng ký/Đăng nhập**: Đăng nhập để có thể tạo bài viết, tương tác vói bài viết
- **Tạo bài viết**: Bài viết được tạo sẽ được kiểm duyệt bằng AI trước khi đăng, người dùng đăng nhập để dùng được chức năng này
- **Bình luận bài viết**: Bình luận sẽ được duyệt bằng AI trước khi đăng, người dùng đăng nhập để dùng được chức năng này  
- **Thích bài viết**: Người dùng đăng nhập để dùng được chức năng này
- **Liên hệ**: Gửi email feedback với admin website, không cần đăng nhập 
- **Quản lý tài khoản**: Đổi thông tin người dùng như: Ảnh bìa, ảnh đại diện, tên hiển thị, lời giới thiệu  

## Hướng dẫn cài đặt  

### Clone dự án  

```bash  
git clone https://github.com/lam-2005/dev_journal.git
```  

### Frontend
```bash  
cd frontend  
```  

### Backend
```bash  

cd backend  
```  

### Cấu hình môi trường  

Tạo file `.env` dựa trên  file mẫu `env.example` 


### Frontend
### Cài đặt dependencies  

```bash  
npm install  
```  

### Chạy Frontend

```bash
npm run build
npm start  
```  

### Backend
### Cài đặt dependencies  

```bash  
npm install  
```  

### Chạy Backend

```bash
npm start  
```
